# Páginas legais do Vinza

Para publicar pelo GitHub Pages:

1. Envie a pasta `docs/` para a branch principal do repositório.
2. Em **Settings → Pages**, escolha **Deploy from a branch**.
3. Selecione a branch principal e a pasta **/docs**.
4. Configure o domínio personalizado, se desejado.

Antes da publicação, confirme que `privacidade@vinza.app` e
`suporte@vinza.app` recebem mensagens. As URLs padrão usadas pelo aplicativo são:

- `https://vinza.app/privacy.html`
- `https://vinza.app/terms.html`

Elas podem ser substituídas no build com `PRIVACY_POLICY_URL` e `TERMS_URL`.
