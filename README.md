# Tynus Site

Site institucional da Tynus, construído em HTML, CSS e JavaScript sem dependências.

## Rodar localmente

```bash
python3 -m http.server 4173
```

Acesse `http://localhost:4173`.

## Publicação

O projeto é compatível com GitHub Pages. A raiz do repositório já contém o
`index.html` e o arquivo `.nojekyll`.
